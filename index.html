I'm Husain, junior undergrad, pursuing dual majors in Electrical and Computer Science Engineering at IIT Gandhinagar. Projects

Image Reconstruction and Super-Resolution

[Mar-Apr 2024]

I enhanced image resolution by combining linear regression with Random Fourier Features (RFF). This approach allowed me to effectively model non-linear image relationships by leveraging RFF for feature engineering. To demonstrate its practicality, I showed the scalability and efficiency of this method in real-world image reconstruction applications.


Next Word Prediction

[Mar-Apr 2024]

I developed and implemented a Multi-Layer Perceptron (MLP) model that integrates word embeddings and deep learning architectures to capture semantic relationships between words. The model was trained on various text corpora and its performance was optimized using techniques such as stochastic gradient descent and activation functions like ReLU and softmax.


Lab Management Web App

[Jan-Mar 2024]

I developed a lab management web app using Flask and MySQL. It organizes lab activities, manages inventory, and handles course administration. For security, I added Google authentication and protected against SQL injection, XSS attacks, and brute-force password hijacking. I optimized database performance with indexing and ensured ACID properties. I also added concurrency support for multi-user access, ensuring data consistency.


HARS-Human Activity Recognition System

[Prof Nipun Batra]

[Jan-Feb 2024]

I developed a Human Activity Recognition (HAR) system using accelerometer and gyroscope data from the UCI-HAR dataset. I preprocessed the data and applied Principal Component Analysis (PCA) and the Time Series Feature Extraction Library (TSFEL) for feature engineering. Comparing raw data to featurized data, I deployed the model on smartphone sensor data, conducting thorough testing and analysis to evaluate its performance, accuracy, and confusion matrices in recognizing human activities.

TESTING RANDOMNESS OF PRNGs

[Prof. Joycee Mekie]

[Jan-May 2023]

I conducted tests on four different Pseudo Random Number Generators (PRNGs): the Physical Unclonable Functions (PUF) based generator, the python-random module generator, the python-secrets module generator, and the web-based (Random.org) generator. To evaluate their performance, I conducted various statistical tests to assess their randomness. I also calculated P-values to determine the utility and reliability of each generator.


Image Classification using CNN on FPGA

[Prof. Joycee Mekie]

[Sep-Nov 2022]

I designed a Convolutional Neural Network (CNN) architecture specifically for image classification using the CIFAR-10 dataset. The architecture included sliding window convolutions, ReLU activation, max pooling, flattening, fully connected layers, SoftMax activation, and weight quantization. To implement the CNN, I used Verilog, a hardware description language, and deployed it on an FPGA. This approach allowed for efficient and hardware-accelerated image classification.
